# Sample Project Documentation

This is a sample project to demonstrate local source aggregation.

## Project Structure

```
sample-project/
├── README.md           # This file
├── CLAUDE.md          # Claude-specific instructions
├── docs/
│   ├── api.md         # API documentation
│   └── setup.md       # Setup guide
└── src/
    └── notes.txt      # Development notes
```

## Overview

This sample project shows how Omni MCP Hub can aggregate documentation from various local files and directories.

## Features

- Automatic discovery of documentation files
- Support for nested directories
- Multiple file format support
